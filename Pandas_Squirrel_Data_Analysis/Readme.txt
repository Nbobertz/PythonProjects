The purpose of this python script is to demonstrate the abiliyt to use the Pandas library to sort through a massive CSV file and isolate values in a desired colum.

Simply run the main file with the other files in the same directory and you will see how many colors there are in all the Squirrels in Central Park in 2018. 

Dataset comes from the 2018 volunteer Squirrel data census: https://data.cityofnewyork.us/Environment/2018-Central-Park-Squirrel-Census-Squirrel-Data/vfnx-vebw
